## Demo Site Overview

The Demo Site showcases modern headless CMS patterns with:

- **24 content models** (6 Content Items, 18 Content Lists)
- **20 component definitions** (Hero variants, content display, interactive components)
- **7 main pages** (Home, About, Pricing, Blog, Features, Contact)
- **2 locales** (English, French)
- **AI-powered search** (Azure OpenAI + Algolia)
- **Personalization** (Audience and region targeting)
- **A/B testing** (PostHog integration)
